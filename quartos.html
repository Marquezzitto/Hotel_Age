Nossas Acomodações

No Age Hotel, todos os nossos quartos são para não fumantes, equipados com TV, Wi-Fi gratuito, mesa de trabalho e banheiro privativo, garantindo uma estadia prática e confortável.

Conheça nossos tipos de quartos:

Quarto Simples

Ideal para viajantes individuais, oferecendo um ambiente funcional e tudo o que você precisa para uma boa noite de sono. Perfeito para quem busca praticidade e um ótimo custo-benefício.

  • Cama de solteiro ou casal (mediante disponibilidade)
  • TV de tela plana
  • Wi-Fi gratuito
  • Mesa de trabalho
  • Banheiro privativo
  • Para não fumantes

A partir de R$ 200 / noite

Reservar Quarto Simples

Quarto Duplo

Conforto e espaço para duas pessoas. Nosso quarto duplo é ideal para casais ou amigos que viajam juntos, com comodidades pensadas para sua conveniência.

  • Cama de casal ou duas camas de solteiro
  • TV de tela plana
  • Wi-Fi gratuito
  • Mesa de trabalho
  • Banheiro privativo
  • Para não fumantes

A partir de R$ 300 / noite

Reservar Quarto Duplo

Quarto Triplo

Para famílias pequenas ou grupos de amigos, o quarto triplo oferece espaço extra e conforto. Desfrute de uma estadia agradável com a praticidade que você merece.

  • Três camas de solteiro ou uma cama de casal e uma de solteiro
  • TV de tela plana
  • Wi-Fi gratuito
  • Mesa de trabalho
  • Banheiro privativo
  • Para não fumantes

A partir de R$ 400 / noite

Reservar Quarto Triplo

Suíte

Nossa suíte oferece um pouco mais de espaço e conforto, com uma pequena área de estar. Perfeita para quem busca um ambiente mais amplo para relaxar após um dia de compromissos.

  • Cama king-size ou queen-size
  • Área de estar separada
  • TV de tela plana
  • Wi-Fi gratuito
  • Mesa de trabalho
  • Banheiro privativo
  • Para não fumantes

A partir de R$ 600 / noite
