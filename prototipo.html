Selo de Qualidade em Canvas

Seu navegador não suporta o elemento canvas.

Este desenho em canvas representa nosso selo de qualidade visual. Ele é gerado dinamicamente pelo navegador, cumprindo o requisito de uso da tecnologia Canvas.

Código JavaScript de Desenho:

O script abaixo desenha o retângulo e o texto no canvas:

                // O JavaScript é o que desenha no canvas
                const canvas = document.getElementById('seloCanvas');
                const ctx = canvas.getContext('2d');
                
                // Desenha o fundo (Retângulo Principal)
                ctx.fillStyle = '#0056b3'; // Azul escuro
                ctx.fillRect(20, 30, 210, 90);
                
                // Desenha a borda amarela
                ctx.strokeStyle = '#FFC300';
                ctx.lineWidth = 4;
                ctx.strokeRect(20, 30, 210, 90);

                // Adiciona o texto do nome do hotel
                ctx.fillStyle = '#FFFFFF'; // Branco
                ctx.font = '24px Arial, sans-serif';
                ctx.textAlign = 'center';
                ctx.fillText('HOTEL AGE', 125, 85);
